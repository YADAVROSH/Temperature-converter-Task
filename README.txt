SkillCraft Technology - Internship Task 01
===========================================

Task Title:
------------
Create a program that converts temperatures between Celsius, Fahrenheit, and Kelvin scales.

Technologies Used:
-------------------
- Java
- Visual Studio Code
- JDK 21

Files Included:
----------------
- TemperatureConverter.java
- Output Screenshot (optional)

How to Run:
------------
1. Open the folder in VS Code or any IDE.
2. Open terminal in the same directory.
3. Compile the file using:
   javac TemperatureConverter.java
4. Run the program using:
   java TemperatureConverter
5. Follow the prompts to enter temperature and conversion type.

Sample Output:
---------------
Enter temperature value: 28  
Convert from (C/F/K): C  
Convert to (C/F/K): F  
Converted temperature: 82.40 F
